Popover

NB: non-modal (unlike dialog which is modal)

  • Popover element (eg a div) has attribute "popover" and an id. It is display:none by default until...
  • Trigger element (eg a button) has attribute "popovertarget" pointing to the id.
  • Clicking outside the popover closes it. But you can also have a popovertargetaction="hide"

Hello, I'm a popover

  • Default position is centre of screen.
  • In css :popover-open styles the popover when shown.
  • CSS Anchor positioning (MDN NB: in Chrome+Edge, not yet in FF or Safari
    • Give the trigger element has an id, give it an "anchor-name" with double dashed name
    • The popover element has position-anchor: --name to tie it together
    • You can then add a top/bottom with the anchor(--name top) css function (add a comma for a fallback)
    • Or use position-area:
    • Chrome canary only: there is an html attribute "anchor"
